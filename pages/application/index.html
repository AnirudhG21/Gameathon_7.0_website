Guidelines

Here are the guidelines for the Gameathon 7.0:

Event Guidelines
  • Each team must consist of 2 - 4 members.
  • The game can be developed using a programming language of your choice.
  • The game must be original and not a copy of an existing game.
  • The game must be submitted using Github Link that consists a zip file containing the game executable and source code.
  • The game must be playable on a Windows 10 operating system.
  • Submission timings will be informed.
  • Important: Please bring your own spike busters to ensure uninterrupted power supply to your systems.
  • System Specifications: To avoid any technical issues, please ensure that your systems meet the following specifications:
    • Processor: Intel Core i5 or equivalent
    • RAM: 8 GB (preferable) or more
    • Operating System: Windows 10 (64-bit) or higher

Application Guidelines
  • No changes/edit to the application is allowed after submitting the application form.
  • Application fee is non-transferable.
  • The declaration must be downloaded from this age from the link provided and must be countersigned, combined and submitted.
  • Individual Declarations are not entertained
  • Payment is only accepted through UPI QR Code present in this portal. No other forms will be entertained.
  • Kindly print the application after payment is done.
  • Uploading combined college ID card PDF is important.
